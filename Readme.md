## 🛠️ Setup and Running

1. Clone this repository:
```bash
git clone https://github.com/yourusername/advent-of-code-2024.git
cd advent-of-code-2024
```

2. Each day's solution can be run independently:
```bash
go run day-01/main.go
# or run tests
go test ./day-01
```
